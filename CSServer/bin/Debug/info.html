Datacentre Alpha Information

What is Datacentre Alpha?


Datacentre Alpha is a Private Remote Access Service Machine which started development in 18/11/2019 to store
logs, data, transmission etc. it uses top-quality SHA512 encryption to encrpyt all
sensitive data such as login details(passwords), access levels(ranks) and other private information.

Datacentre Alpha has 3 ranks in total, ADMIN, MEMBER and GUEST.

GUEST accounts are public access accounts, to log into a guest account just use "guest" for both username and password.
Note however that guest accounts have very limited accessibility.

MEMBER accounts are registered accounts, only trusted users will have ownership of MEMBER accounts, MEMBER accounts have
much more access than GUEST accounts and can access certain data and transmission logs for private usage

ADMIN accounts are private accounts, only organization members have access to admin accounts, admins have full control of
the machine (except for the server console which only exists on the machine's central mainframe) and have access to all data, logs and transmissions.
